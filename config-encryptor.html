⚙ Config Encryptor

Encrypt your Supabase credentials using AES-256-GCM

⚠️ Remember your passphrase — you'll need it to update the config. Never share it.

Setup steps:

1
Encrypt your credentials above and copy the result
2
Create file public/config.enc in your project and paste the string
3
In Vercel dashboard → Settings → Environment Variables, add:
VITE_CONFIG_PASS = your_passphrase
4
Deploy — the app auto-decrypts on load, users never see the raw key
5
To update credentials: re-encrypt here and replace public/config.enc
